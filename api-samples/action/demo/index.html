Action API 演示

尝试更改下面API之前，建议先将扩展固定在工具栏

启用 / 禁用扩展

单击下面的切换启用状态按钮将启用或禁用 Chrome 工具栏和扩展菜单中的扩展操作按钮。

禁用后，单击该操作将不会打开弹出窗口或触发 action.onClicked 事件。

Popup

演示 manifest.json 文件中 action.default_popup 设置为 popups/popup.html。我们能在运行时通过使用 action.setPopup 来修改。

徽标文案

该操作的徽章文本是带有纯色背景色的文本叠加层。这提供了一个被动的 UI 界面来与用户共享信息。它最常用于显示通知计数或当前页面上执行的操作数。

徽标字体颜色

方法 action.setBadgeTextColor 允许更改徽标字体颜色

方法 action.getBadgeTextColor 允许读取当前徽标字体颜色

图标

方法 action.setIcon 允许通过设置图片或原始图片路径的方式改变操作按钮图标 ImageData.

悬停文案

当鼠标悬停在扩展程序的操作按钮上时展示的标题。

在运行时可以使用 action.getTitle 和 action.setTitle 方法来设置和读取扩展悬停的标题
